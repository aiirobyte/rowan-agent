# Version Plans

Last updated: 2026-05-25

Specific version planning lives under:

```text
docs/version/<semver>/
  spec.md
  prompt_plan.md
  todo.md
```

The root files in `docs/` are the current-session entry points. They point agents at the active version plan and keep the cross-session resume flow short.

`docs/PLAN/` is retained as the legacy release-planning tree for historical plans and roadmap context. New or resumed version work should use `docs/version/<semver>/`.
